product 등록

request
  필수
    bucket_id : 버킷ID
    name : 상품명
    date : 날짜
    amount : 상품갯수
    expiration_date : 유통기한
  
  선택
    bar_cd : 바코드
    company : 회사
    memo : 메모
    consump_yn : 소비여부 / 기본값 0
  

response 
  status
    200 : 성공
    400 : 클라이언트 api정보 오류 (필요 request정보 없음)
    422 : 쿼리문 실패 (request 문제 이거나 , query문 잘못됨)
    503 : 과부하 상태 서버 오류 (DB 연결 실패)
